# TSIL Paper Plot Layer

Run commands from the repository root in the `tsil` environment:

```bash
conda activate tsil
export PYTHONPATH="$(pwd):${PYTHONPATH:-}"
```

The paper scripts are parallel to `mt01` and `mt15`. Experiment scripts
stay strict to one experiment folder; paper scripts are flexible and may combine
methods from multiple experiments into one paper figure or table. There is no
main/appendix split here.

## Quick Start

The paper wrappers support `BENCHMARK=mt01` and `BENCHMARK=mt15`. The examples
below set the benchmark explicitly so the command is independent of wrapper
defaults.

Generate all table values:

```bash
BENCHMARK=mt15 bash exec/TSIL/plot/paper/tables/group1_main_method_results.sh
BENCHMARK=mt15 bash exec/TSIL/plot/paper/tables/group2_adaptive_ddl.sh
BENCHMARK=mt15 bash exec/TSIL/plot/paper/tables/group3_sil_analysis.sh
BENCHMARK=mt15 bash exec/TSIL/plot/paper/tables/group4_stability.sh
```

Generate all paper figures as PNG:

```bash
BENCHMARK=mt15 bash exec/TSIL/plot/paper/figures/group1_main_method_results.sh --png
BENCHMARK=mt15 bash exec/TSIL/plot/paper/figures/group2_adaptive_ddl.sh --png
BENCHMARK=mt15 bash exec/TSIL/plot/paper/figures/group3_sil_analysis.sh --png
BENCHMARK=mt15 bash exec/TSIL/plot/paper/figures/group4_stability.sh --png
```

Generate paper figures as PDF:

```bash
BENCHMARK=mt15 bash exec/TSIL/plot/paper/figures/group1_main_method_results.sh --pdf
BENCHMARK=mt15 bash exec/TSIL/plot/paper/figures/group2_adaptive_ddl.sh --pdf
BENCHMARK=mt15 bash exec/TSIL/plot/paper/figures/group3_sil_analysis.sh --pdf
BENCHMARK=mt15 bash exec/TSIL/plot/paper/figures/group4_stability.sh --pdf
```

All figure scripts read `results/TSIL/train_res/...` directly and write
paper artifacts directly. They do not require pre-built raw experiment plots.

## Output Root

All curated paper outputs go under:

```text
results/TSIL/paper_artifacts/
```

Figures are separated by benchmark, group, optional experiment/family, and file
format:

```text
results/TSIL/paper_artifacts/figures/mt01/group_1_main_method_results/compare_temporal/png/
results/TSIL/paper_artifacts/figures/mt01/group_1_main_method_results/ih_reward_sweep/png/
results/TSIL/paper_artifacts/figures/mt01/group_2_adaptive_ddl/compare_temporal/png/
results/TSIL/paper_artifacts/figures/mt01/group_3_sil_analysis/attl/png/
results/TSIL/paper_artifacts/figures/mt01/group_4_stability_exps/png/
```

Tables are separated by benchmark and group:

```text
results/TSIL/paper_artifacts/tables/mt01/group_1_main_method_results/
results/TSIL/paper_artifacts/tables/mt01/group_2_adaptive_ddl/
results/TSIL/paper_artifacts/tables/mt01/group_3_sil_analysis/
results/TSIL/paper_artifacts/tables/mt01/group_4_stability_exps/
```

`paper_metrics.py` table outputs are grouped by file type:

```text
raw/           # per-run CSV files
summary_csv/   # summary CSV files
summary_md/    # summary Markdown tables
```

## What Each Group Reports

Group 1, main method results and motivation:

- Figures: reward-sweep infinite-horizon success curves, main-method success
  curves/summary, and main-method success-time curves/summary. Success-speed
  Pareto plots are available through the training-evaluation plotting task, but
  are not generated by these curated paper scripts.
- Tables: last-10% best success, AUC success, steps-to-80% success, selected
  success episode time, cumulative successful episodes, and reward-sweep
  versions of the same metrics. Successful episode counts use scientific
  notation in summary tables.
- Method set: `compare_temporal` methods plus `PPO_ATTL_TSIL` from
  `compare_tsil`.

Group 2, temporal target analysis:

- Figures: positive-advantage learning-signal maps and four-metric
  learning-signal summaries.
- Tables: fast-success, slow-failure, success-focused, and reward-distracted
  positive advantage mass shares.

Group 3, TSIL/SIL analysis:

- Figures: T30 SIL direction landscape plus curated TSIL/SIL report curves and
  summaries.
- Tables: archive best success time and active positive-gap fast-memory NLL
  measured at the 50% training point.

Group 4, stability experiments:

- Figures: stress-sweep robustness across reward dropout, policy-gradient
  noise, learning rate, and PPO clip ratio.
- Tables: last-10% best success, AUC success, steps-to-80%, successful
  episodes, episode time, first fast-revisit steps, and missing-result notes.

## Common Options

Use another benchmark when that result set exists:

```bash
BENCHMARK=mt15 bash exec/TSIL/plot/paper/tables/group1_main_method_results.sh
BENCHMARK=mt15 bash exec/TSIL/plot/paper/figures/group1_main_method_results.sh --png
```

Use specific task IDs:

```bash
bash exec/TSIL/plot/paper/tables/group1_main_method_results.sh 28
bash exec/TSIL/plot/paper/figures/group1_main_method_results.sh --png 28
```

Use steps or iterations as the x-axis source:

```bash
X_KEY=misc/steps bash exec/TSIL/plot/paper/tables/group1_main_method_results.sh
X_KEY=misc/iterations bash exec/TSIL/plot/paper/tables/group1_main_method_results.sh
bash exec/TSIL/plot/paper/figures/group1_main_method_results.sh --steps --png
bash exec/TSIL/plot/paper/figures/group1_main_method_results.sh --iterations --png
```

For stability, fail if expected results are missing:

```bash
STRICT=1 bash exec/TSIL/plot/paper/tables/group4_stability.sh
```

## What To Modify

Use raw experiment scripts for experiment-specific/debug plots:

```text
exec/TSIL/plot/mt01/
exec/TSIL/plot/mt15/
```

Use paper group scripts for paper composition: which methods, experiments, and
visualizations are included together. For example, group 1 combines `compare_temporal`
baselines with `PPO_ATTL_TSIL` from `compare_tsil`.

Figure scripts call shared plotting helpers through `run_train_curves`,
`run_learning_signal_maps`, TSIL/SIL helpers, or the stability summarizer. Table
scripts call `projects/TSIL/reports/paper_metrics.py`, which is only for
computing table values from training logs. Low-level log loading remains in
`core/plotting/train_data.py`.
